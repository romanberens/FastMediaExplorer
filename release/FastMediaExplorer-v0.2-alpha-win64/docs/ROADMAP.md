# Roadmap

## Milestone A (Done) - Foundation

- Native WinAPI shell.
- Tree + List + Preview.
- Search/filter/sort.

## Milestone B (Done) - Responsive Pipeline

- Async scan + generation tokens.
- Virtual list view (`LVS_OWNERDATA`).
- Thumbnail worker queue + LRU cache.

## Milestone C (Done) - Product Workflows

- Context menu and keyboard actions.
- Overlay/progress/status telemetry.
- Duplicate analysis menu and modes.

## Milestone D (Next) - Duplicate Management

- Grouped duplicate panel.
- Keep-policy engine.
- Recycle-bin-safe batch delete.
- Report export (`CSV/JSON`).

## Milestone E - Performance Scaling

- Multi-thread scan refinement.
- Progressive index.
- Memory envelope controls.
- Smarter thumbnail prefetch.

## Milestone F - Asset Intelligence

- Timeline and statistics panel.
- Largest-files dashboard.
- Metadata extraction (EXIF/resolution/camera).

## Milestone G - Extensibility

- Plugin runtime.
- Watch mode filesystem monitoring.
- Similarity duplicate analyzers.
