# Project Overview

## Product Vision

Fast Media Explorer is a practical desktop tool for fast navigation in large media folders.

The product direction is:

`media viewer -> media browser -> duplicate analyzer -> asset explorer -> lightweight DAM`

## User Problem

Users need to quickly locate the right media files inside large folder trees without waiting for heavy catalog systems.

## Current Scope (`v0.2 alpha`)

- Tree + List + Preview workflow.
- Real-time filtering and sorting.
- Async scan and async thumbnails.
- Duplicate analysis from tools menu.

## Strategic Positioning

Fast Media Explorer sits between:
- Windows Explorer,
- Adobe Bridge,
- Lightroom.

## Non-Functional Goals

- Fast startup.
- UI responsiveness under heavy folders.
- Predictable memory and CPU usage.
- Minimal external dependencies.

## Distribution Model

- Portable release (no installer required):
  - `FastMediaExplorer.exe`
  - `LICENSE`
  - `README` + docs
- Signed binary + checksums.
